# Notebooks Directory

This directory contains Jupyter notebooks for drama video automation:

## Available Notebooks

- **drama_video_creator.ipynb**: Main notebook for creating drama videos with TTS
- **video_editor.ipynb**: Notebook for editing and post-processing videos
- **batch_processor.ipynb**: Process multiple drama scripts at once

## Getting Started

1. Open the notebook in Google Colab or Jupyter
2. Install dependencies from `requirements.txt`
3. Follow the instructions in each notebook
4. Customize parameters as needed
5. Run all cells to generate your video

## Tips

- Use Google Colab for free GPU access
- Save outputs to the `/output` directory
- Check example outputs before running full batches
